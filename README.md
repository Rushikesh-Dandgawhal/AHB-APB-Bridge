# AHB to APB Bridge

## Overview

This project implements an AHB to APB Bridge using Verilog HDL. The bridge enables communication between a high-performance AHB master and low-power APB peripherals.

## Features

* AHB Protocol Support
* APB Protocol Support
* Read Transactions
* Write Transactions
* FSM-Based Controller
* Verilog RTL Implementation
* Functional Verification using Testbench

## Project Structure

* RTL Modules
* Controller Module
* APB Slave MUX
* Testbench
* Simulation Results
* Synthesis Report

## Tools Used

* Verilog HDL
* ModelSim
* Quartus Prime


## Results

The design was successfully simulated and verified for both read and write operations. Synthesis was completed successfully and timing requirements were met.
